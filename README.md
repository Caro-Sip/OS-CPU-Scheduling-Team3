# OS-CPU-Scheduling-Team3

## Projective Objectives

- Understand how CPU scheduling works in an operating system.
- Implement various scheduling algorithms: FCFS, SJF, SRT, RR, and MLFQ.
- Simulate the behavior of these algorithms on a set of user-defined processes.
- Display scheduling results using a Gantt chart, and provide key metrics: Waiting Time, Turnaround Time, Response Time.
- Compare performance between different scheduling strategies

## Setup and installation instructions


for building in environment and run via jvm on class file directly of App.class
```bash
mvn clean compile
```

for installation, create a jar package

```bash
mvn clean package
java -jar target/cpu-1.0-SNAPSHOT.jar
```

## Description of algorithms implemented

- [x] FCFS algorithm whether each processes run consecutively
- [] SJF
- [] SRT
- [] RR
- [] MLFQ

## Instructions on how to run each scheduler

Since the program encompass every algorithms, by running the compiled jar file above, you can choose to use the defautl proccesses or a custom one and choose algorithm to apply the algorithm read the data.
